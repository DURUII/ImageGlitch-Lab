**ImageGlitch** 是一款基于 Web 的故障艺术（Glitch Art）视频生成器，灵感来源于抖音热门话题「给照片通电」以及 Instagram 上的 FlashPhoto 视觉趋势。用户只需上传一张照片，通过简单的点击交互，即可让照片中的元素依次“通电点亮”，生成具有强烈赛博朋克氛围的动态视觉效果。

在社交媒体高度视觉化的当下，人们渴望用更具冲击力的方式分享生活瞬间。「给照片通电」之所以迅速走红，正是因为它打破了传统照片的静态边界，将图像转化为具有未来感和科技感的动态影像。然而，现有制作方式门槛较高：一方面需要使用美图秀秀、醒图等工具进行繁琐的手动抠图；另一方面还依赖剪映等视频软件完成动画与导出，这对普通用户而言成本高、学习曲线陡峭。

ImageGlitch 正是为解决这些痛点而设计。用户只需逐个点击想要点亮的区域，系统可一次性精准分割，传统需要逐个主体抠图的流程链路被大幅简化。同时，产品坚持 **零门槛 Web 化体验**，无需安装任何软件，打开网页即可完成从预览到导出的完整流程。视频导出完全在浏览器端完成，支持 MP4 与 GIF 等格式，避免了服务器排队和算力等待。

在技术实现上，本作品采用以浏览器端实时处理为核心的现代前端架构。前端基于 Next.js 14、React 18 与 TypeScript 构建；AI 能力通过 Transformers.js 在浏览器中加载轻量化的 slimsam-77-uniform 模型，并运行于 Web Worker 内，以确保交互流畅。视频编码与渲染则依赖 FFmpeg.wasm，在前端完成完整的视频生成流程。交互与动画层面使用 Framer Motion 提供平滑的视觉过渡，并结合 @dnd-kit 支持主体顺序的自由拖拽与重排，增强用户对节奏和叙事结构的控制。

此外，ImageGlitch 坚持 **隐私优先** 的设计原则，所有图像、视频及 AI 计算均在本地完成，不上传云端，有效规避隐私与数据合规风险。未来版本将扩展更多故障艺术风格模板，并探索 GUI Agent 驱动的全自动抠图与导出流程，进一步降低创作门槛。
